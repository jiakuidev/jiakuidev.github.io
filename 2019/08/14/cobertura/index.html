类型 含义 通过指标
Line Coverage The percent of lines executed by this test run. 100%
Branch Coverage The percent of branches executed by this test run. 70%
Complexity Average McCabe’s cyclomatic code complexity for all methods. This is basically a count of the number of different code paths in a method (incremented by 1 for each if statement, while loop, etc.) 暂无
N/A Line coverage and branch coverage will appear as “Not Applicable” when Cobertura can not find line number information in the .class file. This happens for stub and skeleton classes, interfaces, or when the class was not compiled with “debug=true.” 不适用
